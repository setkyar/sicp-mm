# SICP မြန်မာဘာသာ (Structure and Interpretation of Computer Programs - Burmese Translation)

ကွန်ပျူတာပရိုဂရမ်များ၏ ဖွဲ့စည်းပုံနှင့် အဓိပ္ပာယ်ဖွင့်ဆိုချက် (SICP) စာအုပ်ကို မြန်မာဘာသာသို့ ဘာသာပြန်ထားခြင်း ဖြစ်ပါသည်။

## About

This is a Burmese (Myanmar) translation of [Structure and Interpretation of Computer Programs, 2nd Edition](https://mitpress.mit.edu/sites/default/files/sicp/full-text/book/book.html) by Harold Abelson, Gerald Jay Sussman, and Julie Sussman.

- **Source**: [sarabander/sicp](https://github.com/sarabander/sicp) (HTML5 version)
- **Translation**: AI-assisted using Google Gemini models, human-reviewed
- **Code examples**: Kept in original English/Scheme — only prose is translated
- **Encoding**: Unicode UTF-8 (not Zawgyi)

## View Online

Open `html/index.xhtml` in your browser, or visit the GitHub Pages site.

## Structure

```
html/           # Translated XHTML files (Burmese)
  css/          # Stylesheets
  js/           # JavaScript
  fig/          # Figures and diagrams
  *.xhtml       # Chapter and section files
source-en/      # Original English source (git-ignored)
scripts/        # Translation pipeline scripts
```
