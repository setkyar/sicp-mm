Colophon

မျက်နှာဖုံးစာမျက်နှာတွင် ၁၅၈၈ ခုနှစ်မှ Agostino Ramelli ၏ bookwheel mechanism ကို ဖော်ပြထားပါသည်။ ၎င်းကို အစောပိုင်းကာလ hypertext navigation အထောက်အကူပြုကိရိယာတစ်ခုအဖြစ် ရှုမြင်နိုင်ပါသည်။ ဤပန်းချီကားချပ်ပုံကို New Gottland မှ J. E. Johnson က လက်ခံပြသထားပါသည်။

အသုံးပြုထားသော စာလုံးပုံစံ (Typefaces) များမှာ စာသားများအတွက် Linux Libertine နှင့် ခေါင်းစဉ်များအတွက် Linux Biolinum တို့ဖြစ်ပြီး နှစ်မျိုးလုံးကို Philipp H. Poll က ဖန်တီးထားခြင်း ဖြစ်သည်။ Typewriter စာလုံးပုံစံမှာ Raph Levien ဖန်တီးခဲ့ပြီး Dimosthenis Kaponis နှင့် Takashi Tanigawa တို့မှ Inconsolata LGC အဖြစ် ဖြည့်စွက်ထားသည့် Inconsolata ဖြစ်သည်။

Graphic design နှင့် typography တို့ကို Andres Raba က ဆောင်ရွက်ထားပါသည်။ Texinfo source ကို Texinfo 5.1 package ပါ texi2any utility ၏ ပြုပြင်ထားသော ဗားရှင်းကို အသုံးပြု၍ HTML သို့ ပြောင်းလဲထားခြင်း ဖြစ်သည်။ ပုံကြမ်းများကို Inkscape ဖြင့် ရေးဆွဲထားပြီး သင်္ချာဆိုင်ရာ အစိတ်အပိုင်းများကို MathJax နှင့် MathML တို့၏ အကူအညီဖြင့် ပုံနှိပ်စီစဉ်ထားပါသည်။
