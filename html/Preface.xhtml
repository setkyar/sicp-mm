ဒုတိယအကြိမ် ထုတ်ဝေခြင်း အမှာစာ

ဆော့ဖ်ဝဲလ်သည် အခြားအရာများနှင့် မတူဘဲ စွန့်ပစ်ရန် ရည်ရွယ်ထားခြင်း ဖြစ်နိုင်ပါသလား- အဓိကအချက်မှာ ၎င်းကို ဆပ်ပြာပူဖောင်းတစ်ခုကဲ့သို့ အမြဲတမ်း ရှုမြင်ရန်ပင် ဖြစ်ပါသလား?

—Alan J. Perlis

ဤစာအုပ်ပါ အကြောင်းအရာများသည် ၁၉၈၀ ပြည့်နှစ်မှစတင်၍ MIT ၏ အခြေခံ ကွန်ပျူတာသိပ္ပံ ဘာသာရပ်အတွက် အခြေခံဖြစ်ခဲ့ပါသည်။ ပထမအကြိမ် ထုတ်ဝေစဉ်က ဤဘာသာရပ်ကို ကျွန်ုပ်တို့ သင်ကြားလာခဲ့သည်မှာ လေးနှစ်ရှိပြီဖြစ်ပြီး၊ ယခု ဒုတိယအကြိမ် ထုတ်ဝေခြင်း မတိုင်မီ ၁၂ နှစ် ထပ်မံ ကြာမြင့်ခဲ့ပါသည်။ ကျွန်ုပ်တို့၏ အလုပ်ကို အခြားသော ကျောင်းသုံးစာအုပ်များတွင် ကျယ်ကျယ်ပြန့်ပြန့် လက်ခံကျင့်သုံးပြီး ထည့်သွင်းဖော်ပြကြသည့်အတွက် ဝမ်းမြောက်မိပါသည်။ ကျွန်ုပ်တို့၏ ကျောင်းသားများသည် ဤစာအုပ်ပါ အယူအဆများနှင့် ပရိုဂရမ်များကို ယူဆောင်၍ ကွန်ပျူတာ စနစ်သစ်များနှင့် ဘာသာစကားသစ်များ၏ အဓိက အစိတ်အပိုင်းအဖြစ် တည်ဆောက်ကြသည်ကို တွေ့မြင်ခဲ့ရပါသည်။ ရှေးဟောင်း Talmudic စကားကစားမှုတစ်ခုကို လက်တွေ့အကောင်အထည်ဖော်သည့်အနေဖြင့်၊ ကျွန်ုပ်တို့၏ ကျောင်းသားများသည် ကျွန်ုပ်တို့၏ တည်ဆောက်သူများ ဖြစ်လာကြပါသည်။ ဤကဲ့သို့ စွမ်းဆောင်ရည်ရှိသော ကျောင်းသားများနှင့် ထူးချွန်သော တည်ဆောက်သူများကို ရရှိသည့်အတွက် ကျွန်ုပ်တို့ ကံကောင်းပါသည်။

ဤထုတ်ဝေမှုအတွက် ပြင်ဆင်ရာတွင်၊ ကျွန်ုပ်တို့၏ ကိုယ်ပိုင်သင်ကြားမှု အတွေ့အကြုံနှင့် MIT နှင့် အခြားနေရာများမှ လုပ်ဖော်ကိုင်ဖက်များ၏ မှတ်ချက်များအရ အကြံပြုထားသော ရှင်းလင်းချက် ရာပေါင်းများစွာကို ထည့်သွင်းထားပါသည်။ ကျွန်ုပ်တို့သည် generic-arithmetic system၊ interpreters၊ register-machine simulator နှင့် compiler အပါအဝင် ဤစာအုပ်ပါ အဓိက ပရိုဂရမ်မင်းစနစ် အများစုကို ပြန်လည် ဒီဇိုင်းဆွဲခဲ့ပါသည်။ ထို့အပြင် IEEE Scheme standard (IEEE 1990) နှင့် ကိုက်ညီသော မည်သည့် Scheme implementation မဆို ဤကုဒ်များကို အသုံးပြုနိုင်စေရန်အတွက် ပရိုဂရမ် ဥပမာအားလုံးကို ပြန်လည် ရေးသားခဲ့ပါသည်။

ဤထုတ်ဝေမှုတွင် အကြောင်းအရာသစ် အချို့ကို အလေးပေးထားပါသည်။ ယင်းတို့အနက် အရေးကြီးဆုံးမှာ computational models များတွင် အချိန်ကို ကိုင်တွယ်ဖြေရှင်းရာ၌ အသုံးပြုသည့် ကွဲပြားသော ချဉ်းကပ်မှုများ၏ အဓိကအခန်းကဏ္ဍ ဖြစ်သည်- objects with state၊ concurrent programming၊ functional programming၊ lazy evaluation နှင့် nondeterministic programming တို့ ဖြစ်ကြပါသည်။ ကျွန်ုပ်တို့သည် concurrency နှင့် nondeterminism ဆိုင်ရာ အခန်းသစ်များကို ထည့်သွင်းထားပြီး၊ ဤအကြောင်းအရာကို စာအုပ်တစ်အုပ်လုံးတွင် ပေါင်းစပ်ထည့်သွင်းရန် ကြိုးပမ်းထားပါသည်။

ဤစာအုပ်၏ ပထမအကြိမ် ထုတ်ဝေမှုသည် ကျွန်ုပ်တို့၏ MIT semester တစ်ခုစာ ဘာသာရပ် သင်ရိုးညွှန်းတမ်းကို အနီးကပ် လိုက်နာခဲ့ပါသည်။ ဒုတိယအကြိမ် ထုတ်ဝေမှုတွင် အကြောင်းအရာသစ်များစွာ ပါဝင်လာသောကြောင့်၊ semester တစ်ခုတည်းအတွင်း အားလုံးကို လွှမ်းခြုံရန် မဖြစ်နိုင်တော့ပါ၊ ထို့ကြောင့် သင်ကြားသူအနေဖြင့် ရွေးချယ်သင်ကြားရပါလိမ့်မည်။ ကျွန်ုပ်တို့၏ ကိုယ်ပိုင်သင်ကြားမှုတွင် တစ်ခါတစ်ရံ logic programming (4.4) အပိုင်းကို ကျော်သွားတတ်ပြီး၊ ကျောင်းသားများကို register-machine simulator အသုံးပြုစေသော်လည်း ၎င်း၏ implementation (5.2) ကိုမူ သင်ကြားခြင်းမပြုပါ၊ ထို့ပြင် compiler (5.5) ကိုလည်း အကြမ်းဖျင်းသာ ခြုံငုံသုံးသပ်တင်ပြပါသည်။ သို့သော်ငြားလည်း၊ ဤသည်မှာ ပြင်းထန်သော သင်တန်းတစ်ခု ဖြစ်နေဆဲပင်။ အချို့သော သင်ကြားသူများသည် ပထမ သုံးခန်း သို့မဟုတ် လေးခန်းကိုသာ သင်ကြားရန် ဆန္ဒရှိနိုင်ပြီး၊ ကျန်ရှိသော အကြောင်းအရာများကို နောက်ဆက်တွဲ သင်တန်းများအတွက် ချန်ထားနိုင်ပါသည်။

World-Wide-Web ဆိုဒ် http://mitpress.mit.edu/sicp သည် ဤစာအုပ်ကို အသုံးပြုသူများအတွက် အထောက်အပံ့ပေးပါသည်။ ယင်းတွင် စာအုပ်ပါ ပရိုဂရမ်များ၊ ပရိုဂရမ်မင်း တာဝန်ပေးချက် နမူနာများ၊ နောက်ဆက်တွဲ အထောက်အကူပြု ပစ္စည်းများနှင့် Lisp ၏ Scheme dialect ကို ဒေါင်းလုဒ်ရယူနိုင်သော implementations များ ပါဝင်ပါသည်။
